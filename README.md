# Dot-to-Dot Book Generator

## Overview
This project provides an automated pipeline to generate dot-to-dot drawing pages from images and assemble them into a printable PDF book. It uses Python with OpenCV, Pillow, NumPy, and ReportLab.

## Prerequisites
- Python 3.11+
- pip

## Installation
1. Clone this repository
2. Install dependencies:
